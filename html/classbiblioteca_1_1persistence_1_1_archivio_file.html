Detailed Description

Gestisce la persistenza dei dati del sistema bibliotecario su file binari. Questa classe si occupa di serializzare e deserializzare gli oggetti principali del modello (libri, utenti, prestiti e credenziali) utilizzando gli stream di oggetti Java.
