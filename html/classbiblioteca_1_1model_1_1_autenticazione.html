Detailed Description

Gestisce la sicurezza e l'autenticazione del bibliotecario.

La classe Autenticazione si occupa di memorizzare e verificare la password di accesso al sistema. Implementa l'interfaccia Serializable per permettere il salvataggio delle credenziali su file (in modo persistente).

Nota sulla sicurezza: La password non viene mai salvata "in chiaro" (plaintext). Viene invece memorizzato solo il suo Hash (impronta digitale), calcolato tramite un algoritmo crittografico (solitamente SHA-256). Questo rende impossibile risalire alla password originale leggendo il file di salvataggio.
