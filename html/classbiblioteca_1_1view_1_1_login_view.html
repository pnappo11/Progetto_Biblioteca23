Detailed Description

Classe che gestisce l'interfaccia grafica di Autenticazione (Login).

Questa classe rappresenta la "View" iniziale dell'applicazione. Si occupa di mostrare il campo per l'inserimento della password e il pulsante di conferma.

◆ LoginView()

biblioteca.view.LoginView.LoginView ( )

Costruttore della LoginView.

Inizializza i componenti grafici e definisce il layout della schermata di login. Viene applicato uno stile CSS inline al bottone di login per evidenziarlo e viene predisposta un'etichetta per la visualizzazione di eventuali messaggi di errore.

◆ setOnLogin()

void biblioteca.view.LoginView.setOnLogin ( Runnable azioneLogin)

Imposta l'azione da eseguire al click del tasto Login.

Il metodo associa al bottone di login una logica di base per la verifica della password inserita. In questa implementazione la password corretta è valorizzata come "admin":

  • se la password è corretta, viene cancellato l'eventuale messaggio di errore e viene invocata la callback fornita in ingresso;
  • se la password è errata, viene mostrato un messaggio di errore e il campo password viene pulito.
Parameters
azioneLoginCallback (Runnable) contenente la logica da eseguire quando l'autenticazione va a buon fine (es. apertura del menu).
